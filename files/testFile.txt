Далее приведено краткое описание усовершенствований, реализованных в версии Java 8:
Методы лямбда-выражений и виртуального расширения
Наиболее примечательная функция Java SE 8 - реализация лямбда-выражений и вспомогательных компонентов для языка программирования и платформы Java.
API-интерфейс даты и времени
Этот новый API-интерфейс позволит разработчикам использовать более естественные и понятные методы обработки даты и времени.
Механизм Nashhorn JavaScript Engine
Новая облегченная высокопроизводительная реализация механизма JavaScript Engine интегрирована в JDK и доступна для приложений Java через существующие API-интерфейсы.
Улучшенная безопасность
Поддерживаемый вручную список методов, чувствительных к вызывающим программам, заменен новым механизмом, который с высокой точностью определяет эти методы и позволяет стабильно обнаруживать их вызывающие программы.
